# Chatting Test
Node.js 및 Socket.io 라이브러리를 활용하여 간단한 안드로이드 채팅 어플 구현

>## 개발환경
><ul>
>  <li>AndroidStudio : 4.1.2</li>
>  <li>API Level min : 16, target : 30</li>
>  <li>Language : Java</li>
>  <li>Node.js : 14.15.1</li>
>  <li>Socket.io : 2.4.1</li>
></ul>

>## 라이브러리
><ul>
>  <li>socket.io</li>
>  <li>gson</li>
></ul>
